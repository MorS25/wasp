Software Overview
=================

Prerequisites 
*************

Inorder to build the system the following software must be installed on your 
computer.

Documentation
-------------
- python-sphinx > 0.6

Onboard Software
----------------
- newlib-arm_1.13_all.deb
- lpc21isp_0.1_i386.deb
- gcc-arm_3.4.4_i386.deb
- binutils-arm_2.16.1_i386.deb

Groundstation
--------------
- python-gtk
- python-serial
